# MyAwesomeConfig

## Configuration
### The default rc.lua file.
A copy of this file is usually installed in /etc/xdg/awesome/rc.lua
### Replace default rc.lua file to current rc.lua file
```
cp /etc/xdg/awesome/rc.lua .config/awesome
```
### Restart AwesomeWM
```
Mod4 + Control + r
```
